## 📞 Контакты и поддержка

### Базовый URL API:
```
https://апи.гостер.рф
```

### Эндпоинты:
- `POST /check` - Проверка документа
- `POST /format` - Форматирование документа

### Требования:
- Формат файла: DOCX
- Максимальный размер: 50 МБ
- Авторизация для `/format`: JWT токен

---

## ✨ Функции в проекте

### ✅ Реализовано:

1. **checkDocument(file)** ✔️
   - Проверяет DOCX на соответствие ГОСТ
   - Без авторизации
   - Возвращает JSON результаты

2. **formatDocument(file, token)** ✔️
   - Форматирует DOCX по ГОСТ
   - С авторизацией (JWT)
   - Возвращает отформатированный DOCX

3. **downloadFormattedFile(blob, fileName)** ✔️
   - Скачивает файл на устройство
   - Добавляет суффикс `_formatted`

4. **Логирование в консоль** ✔️
   - 📤 Отправка запросов
   - ✅ Успешное выполнение
   - ❌ Ошибки
   - 🔐 Авторизация
   - 💾 Скачивание
   - 🚀 Начало процесса

## 🔄 Процесс работы

### Проверка документа:
```
Пользователь → Загружает файл → FreeCheckBlock 
→ checkDocument() → API /check → JSON результаты 
→ Отображение в интерфейсе → Логирование
```

### Форматирование документа:
```
Авторизованный пользователь → Загружает файл → Cabinet 
→ formatDocument() → API /format + JWT → DOCX Blob 
→ downloadFormattedFile() → Скачивание → Логирование
```
