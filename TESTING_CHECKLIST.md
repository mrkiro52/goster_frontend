## 🔐 Авторизация

- [ ] JWT токен хранится в localStorage
  ```javascript
  // В консоли:
  localStorage.getItem('token')  // должен вернуть токен
  ```
- [ ] Токен отправляется в заголовке Authorization
  ```
  Authorization: Bearer {token}
  ```
- [ ] Если токен отсутствует, показывается ошибка
  ```
  ❌ JWT токен не найден. Требуется авторизация.
  ```

## ⚠️ Обработка ошибок

### Тестирование ошибок

- [ ] Загрузить файл без расширения .docx
  - [ ] Ошибка: "Требуется файл в формате DOCX"
  - [ ] Логируется: `❌ Ошибка при проверке документа`

- [ ] Попытаться форматировать без авторизации
  - [ ] Ошибка: "JWT токен не найден"
  - [ ] Логируется: `❌ Ошибка при форматировании документа`

- [ ] Загрузить файл размером более 50 МБ
  - [ ] Сервер должен вернуть ошибку
  - [ ] Логируется: `❌ Ошибка проверки: 413 ...`

## 🌐 Network запросы

### Точка проверки: DevTools Network вкладка (F12)

- [ ] POST запрос к `https://апи.гостер.рф/check`
  - [ ] Status: 200 (успех)

## 📞 Если что-то не работает

1. **Проверьте консоль:** F12 → Console
2. **Проверьте логи:** Ищите 📤, ✅, ❌ эмодзи
3. **Проверьте Network:** F12 → Network (POST запросы)
4. **Проверьте localStorage:** 
   ```javascript
   localStorage.getItem('token')
   ```
5. **Проверьте API доступность:** https://апи.гостер.рф

## 📝 Примечания

- API требует DOCX файлы (не PDF)
- Максимальный размер файла: 50 МБ
- Проверка занимает: 5-10 сек
- Форматирование занимает: 15-30 сек
- JWT токен хранится в localStorage с ключом `token`
