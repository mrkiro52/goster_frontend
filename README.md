# Frontend (лендинг + кабинет)

Клиентская часть на `Vite + React`.

## Запуск

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
npm run preview
```

## Что реализовано

- Лендинг с CTA и блоком "Как это работает"
- Кнопки `Войти` / `Зарегистрироваться`
- Страница авторизации (без отправки данных)
- Переход в личный кабинет
- Личный кабинет:
  - вкладка "Мои документы" (входной/выходной файл, дата и время, иконки)
  - вкладка "Загрузить файл" (файл, выбор ГОСТ, запуск обработки)
  - анимированное пошаговое выполнение обработки ГОСТ 7.32
