## 🎯 Основные эндпоинты

### POST `/check` - Бесплатная проверка
```
URL: https://апи.гостер.рф/check
Метод: POST
Тело: multipart/form-data (файл)
Авторизация: не требуется
Ответ: JSON { score, errors, warnings, ... }
```

### POST `/format` - Форматирование
```
URL: https://апи.гостер.рф/format
Метод: POST
Заголовок: Authorization: Bearer {JWT_TOKEN}
Тело: multipart/form-data (файл)
Авторизация: обязательна JWT
Ответ: DOCX файл (Blob)
```

## 🚀 Использование

### Для пользователя:

**Проверка документа:**
1. На главной странице → "Проверить диплом бесплатно"
2. Загрузить DOCX файл
3. Нажать "Проверить документ"
4. Увидеть результаты

**Форматирование документа:**
1. В личном кабинете → вкладка "Загрузить файл"
2. Загрузить DOCX файл
3. Нажать "Начать форматирование"
4. Файл скачается автоматически

### Для разработчика:

```javascript
import { checkDocument, formatDocument, downloadFormattedFile } from './config/api';

// Проверка
const result = await checkDocument(file);

// Форматирование
const formatted = await formatDocument(file, token);
downloadFormattedFile(formatted, fileName);

// Просмотр логов
// F12 → Console → смотрите логи с эмодзи
```
